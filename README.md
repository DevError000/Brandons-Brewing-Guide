## Features:
- **_Brewing Stand UI Change_**

_This pack will add little GUI elements to the side of the GUI in the Brewing Stand._

   It's not much but I guarantee that this pack **_will_** help you improve your brewing skills, _probably... possibly... maybe?_ **I don't know!** **It will if you use it right!**

## The way it works.
  This pack works because of a feature within Minecraft: Java Edition's Code that supports the ability to add things to the sides of GUI Screens, such as the Chest or Brewing Stand. This was _not_ made to make modders life easier, it was made because some of the GUI Screens in the Vanilla Game use this function, such as the pause screen.
